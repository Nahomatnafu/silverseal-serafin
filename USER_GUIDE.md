# Silverseal Guard Management - User Guide

## Quick Start

The Silverseal Guard Management Tool helps you quickly see and manage your security staff across all client sites.

## Two Main Views

### 🗺️ Map View (Default)
**Best for**: Visual overview of all sites and quick status checks

**What you see**:
- Each site appears as a pin on the map
- Numbers on each pin show:
  - **Green dot + number**: Staff currently on shift
  - **Blue dot + number**: Upcoming shifts (next 24 hours)
  - **Red dot + number**: Staff with certification issues

**How to use**:
1. Click any site pin to see details
2. A panel slides in from the right showing:
   - Site name, address, and client
   - All staff currently on shift
   - All upcoming shifts
3. Click any staff member to see their full profile

### 📋 Roster View
**Best for**: Finding specific staff, filtering by criteria, checking certifications

**How to use**:
1. Click "Roster View" button in the top right
2. Use the search box to find staff by name
3. Use filters to narrow down:
   - **Shift Status**: On shift now, upcoming, or unassigned
   - **Site**: Filter by specific location
   - **Client**: Filter by client organization
   - **Certifications**: Show only staff with valid, expiring, or expired certs
   - **Role**: Filter by guard, bodyguard, supervisor, etc.
4. Sort the list by:
   - Name (alphabetical)
   - Shift start time (earliest first)
   - Certification expiry (soonest first)
5. Click any row to see full staff profile

## Staff Profile Drawer

When you click on any staff member, a detailed profile slides in from the right.

**What you see**:
- **Photo and basic info**: Name, role, status
- **Contact**: Email and phone (click to email/call)
- **Current assignment**: Where they are right now, shift times
- **Certifications**: All certs with color-coded status
  - 🟢 Green = Valid (more than 30 days)
  - 🟡 Yellow = Expiring soon (within 30 days)
  - 🔴 Red = Expired
- **Documents**: Contracts, licenses (click to view)
- **Notes**: Any special notes about this staff member

**To close**: Click the X in the top right corner

## Understanding Status Indicators

### Shift Status
- **Active** (green badge): Currently on shift right now
- **Upcoming** (blue badge): Shift starts within 24 hours
- **Unassigned**: No current or upcoming shifts

### Certification Status
- **Valid**: All certifications are current (30+ days until expiry)
- **Expiring Soon**: One or more certs expire within 30 days
- **Expired**: One or more certs have already expired

## Common Tasks

### "Who's working at Site X right now?"
1. Use Map View
2. Click the site pin
3. See "On Shift Now" section

### "Which staff have expiring certifications?"
1. Switch to Roster View
2. Set Certification filter to "Expiring Soon"
3. Sort by "Cert Expiry" to see soonest first

### "Who's available (unassigned)?"
1. Switch to Roster View
2. Set Shift Status filter to "Unassigned"

### "Find all bodyguards at Metro Bank sites"
1. Switch to Roster View
2. Set Client filter to "Metropolitan Bank Group"
3. Set Role filter to "Bodyguard"

### "See all upcoming shifts"
1. Switch to Roster View
2. Set Shift Status to "Upcoming"
3. Sort by "Shift Start" to see chronological order

## Tips for Efficient Use

1. **Use Map View for quick visual checks** - Great for morning briefings
2. **Use Roster View for detailed searches** - Better for planning and compliance
3. **Watch for red dots on the map** - These indicate certification issues that need attention
4. **Check expiring certifications weekly** - Yellow badges mean action needed soon
5. **Click through to profiles** - Full details are always one click away

## Color Guide

- 🟢 **Green**: Active, on shift, valid certifications
- 🔵 **Blue**: Upcoming shifts, future assignments
- 🟡 **Yellow**: Warning, expiring soon (30 days)
- 🔴 **Red**: Alert, expired certifications, needs attention
- ⚪ **Gray**: Inactive, unassigned, no status
